Font Sizing

Fonts are sized using REMs. Root font size is set to 10px (1rem) and all other elements are sized relative to that (so 1.2rem == 12px, 2.4rem == 24px etc).

Line heights are specified in decimal form (so 1.2 == 120%) with unit omitted (as this ensures child elements will inherit the raw number value, rather than the computed value on the parent element).

Default body font size is 14px (1.4rem).

Lists

No list style

Removes default list styling (padding-left, margin, list bullets).

  • Item one
  • Item two
  • Item three
<ul class="no-list-style">
  <li>Item one</li>
  <li>Item two</li>
  <li>Item three</li>
</ul>

Navigation style

Removes default list styling and makes lis display inline-block.

Elements with the (optional) class .nav-item will display block with a bit of padding so links have more surface area, can have background colors applied etc.

  • Item one
  • Item two
  • Item three
  • Item one
  • Item two
  • Item three

  <li><a href="#" class="nav-item">Item one</a></li>
  <li><a href="#" class="nav-item">Item two</a></li>
  <li><a href="#" class="nav-item">Item three</a></li>
</ul>
