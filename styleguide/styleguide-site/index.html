Color Variables

The following sass variables can be used throughout stylesheets:

Variable Hex Value Swatch
$white #FFFFFF
$black #333333
$grey-dark #BBBBBB
$grey #D4D4D4
$grey-light #EEEEEE
$grey-extra-light #F5F5F5
$orange #F27C00
$red #FF0000

Primary Color Variable

Variable called $primary-color is used for default link color etc. Change this to be the dominant brand color throughout the project (or whatever color you want the links to be).

Variable Hex Value Swatch
$primary-color #F27C00
